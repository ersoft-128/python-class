1. Install django in our system -> pip install django -> type in terminal.

2. Now make a django project. -> type in terminal -> py -m django startproject PakaRubayet -> press Enter.

3. Let's check the server -> type in terminal -> py manage.py runserver -> press Enter.

4. Let's make a app -> type in terminal -> py manage.py startapp app-> press Enter.

5. Let's pipe line the app and project url and views -> go project urls.py -> copy the code -> go app -> make a file name with urls.py also -> paste here the copied code.

6. Go project urls.py -> ensure that import has path and include like (from django.urls import path,include).

7. make secound path include like (path("",include('app.urls')))

8. go app views.py -> and make a funcation for homepage

homepage funcation code:
def homepage(request):,
return render(request,"index.html"),

9. go app urls.py -> type the path like ( path("",homepage,name='home'),)

10. save all the file and go to the terminal-> lets type -> py manage.py runserver-> press the enter.
